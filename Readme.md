# FloorDraw

FloorDraw is a Javascript tool for drawing polygons and store into json. and read back.
* [D3JS] [Pldjs]
 

# Installation
First you can download  ``http-server`` from [NPM] [HtpSrv]. 

Then run this command for view on browser
```sh
http-server -p 8000
```
